1) run app.py from terminal
>> python app.py

2) Go to browser and hit an url - http://localhost:5000/

3) You should get a form, just hit submit and you will get a prediction form deep learning module.

4) To change model to randomForestClassifier you have to edit a url value in index.html [ i.e. replace action = "/dlcheckFraud" by action = "/checkFraud" ]
